Configuration Reference
========================

Configuration files, command-line options, and environment variables for RiESCUE tools.

:py:class:`riescue.FeatMgr` is the Python dataclass used to manage the configuration of the test. It can be directly passed into ``RiescueD`` when using the Python API.
See the :doc:`FeatMgr <../python_api/FeatMgr>` reference for more information.
