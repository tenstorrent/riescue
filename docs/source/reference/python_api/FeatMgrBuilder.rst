FeatMgrBuilder
---------------
``FeatMgrBuilder`` is a builder class for ``FeatMgr``. It allows for the construction of a ``FeatMgr`` object from multiple sources.
It is used to construct a ``FeatMgr`` object from test header, a cpu config, and command line arguments.
